## Requirements

**Python** >= 3.6

Update your PYTHONPATH env variable for instance by editing your **.profile** home file:
```
  PYTHONPATH=$PYTHONPATH:$HOME/pre.di.c/bin:$HOME/pre.di.c/clients/bin
```
